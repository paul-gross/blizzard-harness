# Verification — blizzard

Blizzard's **verifiability matrix** — the inventory of concrete methods a skill or agent uses to assert a blizzard change is correct.
Conforms to the canon concept at `winter-canon:/verifiability-matrix.md`.

An agent reads the matrix whenever it plans how a change will be checked, not only at review time — so a plan already names how it will be proven, or that the proof mechanism must be built first.
